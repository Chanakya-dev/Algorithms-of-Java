## **N-Queens Problem**

### **Problem**

Place **N queens** on an **N×N chessboard** such that no two queens attack each other.

* **Queens attack** in:

  * Same **row**
  * Same **column**
  * Same **diagonal** (both left and right diagonals)

* **Goal:** Find **all possible arrangements** of N queens on the board.

---

### **Example**

For `N = 4`, one solution is:

```
. Q . .
. . . Q
Q . . .
. . Q .
```

Another solution is:

```
. . Q .
Q . . .
. . . Q
. Q . .
```

---

### **Key Concepts**

1. **Backtracking** – Place queens **row by row**.
2. **Recursive Approach** – For each row, try placing a queen in every column and check if it’s **safe**:

   * No other queen in the same column
   * No other queen in the **upper-left diagonal**
   * No other queen in the **upper-right diagonal**
3. **Base Case** – If all rows are filled, a solution is found.

---

### **Step-by-Step Approach**

1. Start at **row 0**
2. Try placing a queen in **each column** of the current row.
3. For each valid placement, **recurse to the next row**.
4. If no column is valid → **backtrack** to previous row and try the next column.
5. Continue until all rows are filled → print/store the board.

---

### **Code Example (Java)**

```java
import java.util.*;

public class NQueens {
    public static void solveNQueens(int n) {
        int[] board = new int[n]; // board[i] = column index of queen in row i
        placeQueens(board, 0, n);
    }

    private static void placeQueens(int[] board, int row, int n) {
        if (row == n) {
            printBoard(board, n);
            return;
        }

        for (int col = 0; col < n; col++) {
            if (isSafe(board, row, col)) {
                board[row] = col; // place queen
                placeQueens(board, row + 1, n); // recurse
                // backtrack happens automatically by overwriting board[row]
            }
        }
    }

    private static boolean isSafe(int[] board, int row, int col) {
        for (int i = 0; i < row; i++) {
            if (board[i] == col || Math.abs(board[i] - col) == row - i) {
                return false; // same column or diagonal
            }
        }
        return true;
    }

    private static void printBoard(int[] board, int n) {
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                if (board[i] == j) System.out.print("Q ");
                else System.out.print(". ");
            }
            System.out.println();
        }
        System.out.println();
    }

    public static void main(String[] args) {
        int n = 4;
        solveNQueens(n);
    }
}
```

---

### **Key Points**

1. Uses **backtracking**: try → recurse → backtrack.
2. **Safety check** ensures no queens attack each other.
3. Time Complexity: O(N!) → because each row can potentially place N queens, decreasing choices in subsequent rows.
4. Space Complexity: O(N) → storing column positions in a 1D array.
5. Variants asked in interviews:

   * Count total solutions instead of printing
   * Solve for **N > 8** efficiently
   * Print solutions in **lexicographical order**
